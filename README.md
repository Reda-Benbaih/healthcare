Healthcare Management API
📌 Project Description

This project is a RESTful API developed using Spring Boot for managing a healthcare system.
It allows the management of patients, doctors, appointments, and medical records in a structured and efficient way.

🎯 Objectives
Manage patients and doctors
Schedule and track appointments
Store and retrieve medical data
Provide a scalable backend for healthcare systems
🛠️ Technologies Used
Java 17+
Spring Boot
Spring Data JPA
Hibernate
MySQL / PostgreSQL
Maven
Flyway (Database Migration)
📂 Project Structure
src/
 ├── controller       # REST Controllers
 ├── service          # Business Logic
 ├── repository       # Data Access Layer
 ├── model            # Entities
 ├── DTO              # Data Transfer Objects
 ├── mapper           # MapStruct / Manual Mappers
